In the movie data set there are 10866 records and 21 columns. Each movie has an id that uniquely identifies it. It also have 
imdb_id the column is an identification from another ranking database,
popularity which ranks it from highest to lowest
budget the amount scheduled in movie creation
revenue the amount collected
original_title the name of the movie
cast the featuring actors
homepage the column ontains website url
director the column contains the name movie director
tagline the column inclusive words
overview the column contains summary of movie
runtime the column contains the time to play the movie
genres the column contains the categories where the movie belongs
production_companies the column conatins list of companies in movie production
release_date the column contains the date the movie will launch
vote_count the column contains the number of fans votes
vote_average the column contains the average votes
release_year the column contains the year to release the movie
budet_adj the column contains the budet in exponential format
revenue_adj the column contains the amount in exponential form
